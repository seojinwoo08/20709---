# 매점 상품 현황 관리 시스템

토평고등학교 매점의 상품 현황을 온라인으로 확인할 수 있는 시스템입니다.

## 주요 기능

- 📦 오늘의 매점 상품 현황과 가격 확인
- 🔔 품절/입고 알림 기능
- 📋 성분표 확인 기능
- 📊 인기상품 순위 및 판매 통계
- 🔍 카테고리별 정렬 및 검색

## 설치 방법

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py
```

## 사용자

- 토평고등학교 학생
- 토평고등학교 선생님

## 주요 모듈 역할 및 설명

### 1. Streamlit (st)
**역할**: 웹 애플리케이션 프레임워크로, Python 코드만으로 인터랙티브한 웹 인터페이스를 구성합니다.

**구체적 기능**:
- `st.title()`, `st.header()`: 페이지 제목과 섹션 제목 표시
- `st.sidebar`: 사이드바를 통한 메뉴와 필터 제공
- `st.columns()`: 다중 컬럼 레이아웃으로 상품 카드 그리드 구성
- `st.selectbox()`, `st.radio()`: 사용자 입력을 받는 UI 컴포넌트 제공
- `st.button()`: 버튼 클릭 이벤트 처리
- `st.dataframe()`: pandas 데이터프레임을 표 형식으로 표시
- `st.info()`, `st.warning()`, `st.success()`: 알림 및 상태 메시지 표시
- `st.session_state`: 사용자 세션 상태 관리 (선택된 상품, 로그인 상태 등)

### 2. Pandas (pd)
**역할**: 데이터 조작 및 분석을 위한 라이브러리로, CSV 파일의 상품 데이터를 효율적으로 처리합니다.

**구체적 기능**:
- `pd.read_csv()`: CSV 파일에서 상품 데이터를 데이터프레임으로 읽어오기
- `pd.DataFrame()`: 데이터를 구조화된 테이블 형태로 관리
- `.sort_values()`: 상품을 판매량, 가격, 이름 등으로 정렬
- `.groupby()`: 카테고리별로 데이터를 그룹화하여 통계 계산
- `.loc[]`: 특정 조건에 맞는 행을 선택하여 재고 업데이트, 판매량 증가 등 데이터 수정
- `.iterrows()`: 데이터프레임의 각 행을 순회하여 상품 카드 생성
- `pd.DataFrame.to_csv()`: 수정된 데이터를 CSV 파일로 저장

### 3. Plotly Express (px)
**역할**: 인터랙티브한 데이터 시각화 라이브러리로, 판매 통계를 그래프로 표현합니다.

**구체적 기능**:
- `px.bar()`: 인기 상품 TOP 10의 판매량을 가로 막대 그래프로 표시
  - `orientation='h'`: 가로 방향 막대 그래프 생성
  - `labels`: 축 레이블 한글화 (판매량, 상품명)
- `px.pie()`: 카테고리별 판매량 비율을 파이 차트로 표시
  - `values='총 판매량'`: 파이 차트의 크기 기준
  - `names='카테고리'`: 각 파이 조각의 이름
- `.update_layout()`: 그래프의 레이아웃 설정 (y축 정렬 등)
- `st.plotly_chart()`: Streamlit에 그래프를 삽입하여 사용자가 마우스로 상호작용 가능 (확대, 축소, 호버 정보 표시)
